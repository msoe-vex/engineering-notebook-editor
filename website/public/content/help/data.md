# Import & Export

Easily move your data between different workspace modes or back it up.

## Export Notebook
Download everything (entries, images, team data) as a single JSON file. We recommend doing this at the end of every meeting as a "hard backup".

## Import Notebook
Upload a previously exported JSON file to restore your work into any workspace. This is useful if you want to switch from a Temporary workspace to a GitHub one.

## Entry Export
You can also export individual entries by right-clicking them in the sidebar. This is great for sharing specific parts of your work with other teams or for backup.
