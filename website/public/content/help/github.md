# GitHub Collaboration

Syncing with GitHub allows multiple team members to work on the notebook while keeping a full history of every change.

## Staging Changes
When you save an entry, it is stored locally in your browser first. These are called "Pending Changes". You can see them in the sidebar with a yellow indicator.

## Pushing to GitHub
Use the "Push to GitHub" button in the sidebar to upload all your pending changes to the repository at once. This creates a "commit" on GitHub.

## Pulling Changes
When you open a GitHub workspace, the editor automatically fetches the latest changes from the repository. If you have been working on a different computer, your work will be synced automatically.
